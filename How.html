How to Play CODEL

Welcome to CODEL! Every day, you'll be challenged with a new coding task. Whether you're a beginner or an experienced coder, this game will help you practice and improve your Python skills.

Objective

The goal of the game is to solve the challenge in either the least amount of time or the least number of attempts. For example:

  • A solve with 2 attempts and 10 minutes would beat a solve with 3 attempts and 5 minutes.
  • However, a solve with 2 attempts and 5 minutes would beat a solve with 2 attempts and 10 minutes.

The Basics

1. Daily Challenge

Every day, a new coding challenge will be available. The challenge will appear right below the CODEL heading, so you'll always know what you're working on.

2. Write Your Code

Below the challenge description, you'll find an input box where you can write your Python code to solve the problem. Use your best coding skills to write a solution. Don't worry if you're unsure; you can always test your code before submitting it!

3. Test Your Code

Once you're happy with your code, you have two options:

  • Run Code: If you'd like to test your solution yourself, click the Run Code button. This will run your code and let you see the output right away.
  • Submit Code: If you're confident your code is correct, click the Submit Code button to submit it for testing.

4. See How You Did

After you submit your code, the game will run a series of tests to check if your solution is correct.

  • If everything works perfectly, you'll see green squares next to each test — indicating that you passed all the tests.
  • If any tests fail, you'll see red squares. These indicate where your solution didn't work as expected.

5. What Happens if There’s an Error?

If there is an error in your code (such as a syntax error or an exception), it won't count as a submission. Instead, you'll see an error message telling you what went wrong. Don’t worry, just fix the error and try again!

Tips for Success

  • Test First: If you're unsure whether your solution works, hit Run Code to check it before you submit.
  • Read the Instructions Carefully: Make sure you understand the challenge before you start coding. The more clearly you understand the problem, the easier it will be to solve.
  • Don’t Rush: Take your time to write clean, efficient code. You have plenty of time to work on each challenge, so no need to rush.
  • Check Your Output: When running your code, check the output to make sure it matches what’s expected. If the output is wrong, adjust your code and test again.

Ready to Start?

You’re now ready to start coding! Click 'BACK TO CODEL' in the top bar to begin and see what’s waiting for you today. Good luck, and happy coding!
